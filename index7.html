img is the name of the tag used to create images in the web page.
img tag has four properties that we must include in order to proper set the image.
the first one is src - src is used for telling the img tag what is the name of the file that contains the image.
the second one is called alt - alt is used for telling the img tag what is alternative as a name for the
image in case the image file is missing.
the thrid one is width- width is used for to tell how wide you want the picture to be when you upload it.
the last one is called height = height is meant for telling how tall the picture size you wanted to be.
